Projeto para universidade

Cofre Automático

Projeto para universidade onde usando uma placa arduino uno, um servo motor, um LCD com módulo I2C, 5 botões, um buzzer e um módulo wifi ESP8266. O sistema foi montado em uma caixa de sapato e pode armazenar qualquer objeto de pequeno porte. A programação embarcada foi feita em C++.
