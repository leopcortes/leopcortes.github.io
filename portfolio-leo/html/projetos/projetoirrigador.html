Projeto para universidade

Irrigador Automático

Projeto para universidade onde usando uma placa arduino uno, uma bomba peristáltica e um sensor de umidade do solo. O sistema serve para regar plantas residenciais pequenas e só precisa de uma fonte de alimentação e uma fonte de água (copo ou garrafa d'água). O sistema ainda conta com led's indicativos que mostram se o solo está úmido, seco ou sendo irrigado. A programação embarcada foi feita em C++.
