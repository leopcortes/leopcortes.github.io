Projeto pessoal

Calculadora Avançada

Calculadora avançada desenvolvida para por conhecimentos de HTML, CSS e JavaScript em prática. Projeto feito com o proposito de ser visualmente simples, intuitivo e rápido. Conta com diversos tipos de calculadora: calculadora básica de celular, calculadora de horários da UnB, calculadora científica e de sistemas, calculadoras de conversões de unidades de medida, calculadoras de associação de componentes eletrônicos, entre outras.
